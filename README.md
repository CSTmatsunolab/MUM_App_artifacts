# MUM_App_artifacts

本リポジトリは、MUM（学習支援アプリ）の**研究成果物（ドキュメント）**を公開するためのものです。  
※ アプリのソースコード、鍵・証明書・アカウント情報、内部データ等は含みません。

## 含まれる成果物（docs/）
- iOSリリース手順書: docs/release/ios_release.md
- Androidビルド・提出手順書: docs/release/android_release.md
- プロジェクト構成（構造説明）: docs/architecture/project_structure.md
- 更新履歴・開発データ: docs/updates/release_history.md

## 注意
- 公開にあたり、機微情報（鍵・証明書・アカウント情報・ログ等）はリポジトリに含めません。
