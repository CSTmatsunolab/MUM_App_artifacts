## 3. iOS リリース履歴

### 3.1 バージョン 1.0（初回）

- 公開日：2025年11月5日

### 3.2 バージョン 1.0.1

#### 時間割ページ

- 年度・学期が変わるごとに履修を変更できるように修正
- 履修登録できる授業が学年に対応されるように修正
- 授業詳細画面に校舎と単位の情報を追加
- 授業詳細画面における教員／教室／校舎／単位の情報表示を左揃えに統一
- ライトテーマ時のみ共有ボタンが見えにくい不具合を修正

#### マップページ

- 授業検索の際に駿河台校舎が船橋校舎と判別される不具合を修正
- ポップアップ表示における授業／教室／校舎／教員の4つを太字で表示し，情報が「：」より右に表示されるように修正
- ポップアップ表示における×ボタンの設置場所を右から右上に変更

### 3.3 バージョン 1.0.2

#### 時間割ページ

- 時間割表から授業を登録する際に検索バーを追加
- 授業登録時に登録確認画面を追加
- 科目名と設定ボタンおよび×ボタンが被らないように修正
- アプリを閉じても設定した年度がリセットされないように修正

### 3.4 バージョン 1.1.0

#### 時間割ページ

- 共有ボタンが押せない不具合を修正
- 学年および年度を選択するモーダルに飛べない不具合を修正
- 年度／学期切替ページのUIを修正
- ヘッダーを削除
- 初期画面で時間割表が大きく表示されるように変更
- 共有ボタンをアイコンに変更
- 土曜日表示／非表示切替機能を追加
- 設定ボタンに「年度／学期ボタン」「土曜日を表示スライドボタン」を配置
- ダークテーマに対応していなかったカードやモーダルをダークテーマに対応できるように修正

#### その他ページ

- その他タブのアイコンを変更
- 利用者アンケートのURLを変更
- テーマ設定でデバイスのテーマを使用できるように変更
- テーマ設定モーダルを変更

## 4. Android リリース履歴

### 4.1 クローズドテスト期間

- 1.0.0（2025年7月30日）：クローズドテスト開始
- 1.0.1（2025年10月15日）：設定タブ新設（テーマ切替等），授業詳細画面のUI改善，ToDo表示修正等
- 1.0.2（2025年10月16日）：内部依存関係の整合性修正，起動安定性改善等
- 1.0.3（2025年10月23日）：時間割UI改善，不具合修正，マップ下限設定，アンケート導線追加
- 1.0.4（2025年11月14日）：時間割テンプレート改善，マップ操作性改善，ヘッダー／タブ表示改善
- 1.0.5（2025年11月15日）：学期判別不具合修正（選択学期のみ登録可能に）

### 4.2 リリース後

- 1.0.6（2025年11月24日）：製品版リリース開始（時間割・マップ・ToDoの改善を反映）
- 1.0.7（2025年12月11日）：時間割検索バー，登録確認画面，UI修正
- 1.0.8（2025年12月19日）：年度／学期UI改善，土曜日表示切替，ダークテーマ対応等
